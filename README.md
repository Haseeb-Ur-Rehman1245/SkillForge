⚒️ **SkillForge – Interactive Developer Learning Platform**

SkillForge is a modern, interactive roadmap platform designed to help developers learn smarter, not harder.

🚀 It transforms traditional learning paths into a **visual, engaging experience** where users can explore, track, and master skills step-by-step.

✨ **Key Features**

* 🧭 Structured roadmaps for **Frontend, Backend & DevOps**
* 🎯 Interactive nodes with **progress tracking**
* 🔓 Smart unlocking system for guided learning
* 🌌 Animated UI with immersive developer-focused design
* 💾 Persistent progress using localStorage
* 🌗 Light/Dark mode for better user experience


🧠 **Why SkillForge?**

Most learning platforms are static and overwhelming.
SkillForge makes learning:

* Visual 👀
* Interactive ⚡
* Goal-driven 🎯

🛠️ **Built With**

* HTML, CSS, JavaScript
* Canvas API (animated background)
* LocalStorage for state management

To build a developer learning ecosystem where progress feels like a game and growth is clearly visible.
